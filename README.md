### Prequisites

- Install [tuckr](https://github.com/RaphGL/Tuckr#installation)

### Installation

> [!CAUTION]
> _**Warning**: This will overwrite your existing dotfiles_

For example, to install the `hyprland` dotfiles:

```bash
tuckr add hyprland
```

To check status of installed dotfiles:

```bash
tuckr status
```
